# 2D-Movement--Unity--platformer
<br>
<p>THis is a Unity Script for 2D platformer where the player can jump as many times as we want, resets it's jumps when collides with anything</p>
**<p>Moves</p>**
<br>
jump
<br>
walk (W A S D)
<br>
run (Left Shift + W A S D)
<br>
crouch (Ctrl + W A S D)
